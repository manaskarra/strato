## 🚀 Quick Start

1. **Open Learning Lab**
   - Navigate to `/learning` in your app
   - Or click "Learning Lab" in the sidebar

2. **Try a Template**
   - Click "Templates" button
   - Select "AI Research Workflow"
   - See the pre-built workflow

3. **Build Your Own**
   - Drag "Start" node to canvas
   - Add analysis nodes
   - Connect them by dragging from right handle to left handle
   - End with "End" node

4. **Learn**
   - Click the "✨ Learn" button
   - Get personalized tutorial
   - Take quiz to test understanding

## 🎨 Visual Features

### Node Styling
- **Workflow nodes** (Start/End): Animated pulse, prominent borders with green theme
- **Input nodes**: Blue theme (#3b82f6)
- **Analysis nodes**: Pink theme (#ec4899)

### Interactions
- **Drag & Drop**: Drag tools from sidebar to canvas
- **Connect**: Drag from right handle → left handle
- **Move**: Drag nodes to reposition
- **Zoom**: Use controls or mouse wheel
- **Delete**: Select node and press Delete/Backspace

### Animations
- Hover effects on tool cards
- Pulse animation on workflow nodes
- Animated connections between nodes
- Smooth transitions

## 🔧 Tips & Tricks

1. **Start Simple**: Begin with 3-4 nodes, add complexity gradually
2. **Use Templates**: Learn patterns from pre-built workflows
3. **Save Workflows**: Save your favorite workflows for reuse
4. **Experiment**: Try connecting nodes in different ways
5. **Learn Mode**: Always click "Learn" to understand what you built

## 🎯 Node Categories Explained

### WORKFLOW
Control the flow of your analysis
- **Start** - Begin your workflow
- **End** - Complete your workflow

### INPUT
Where your data comes from
- **Stock Symbol** - Enter a specific ticker
- **Stock Selection** - Choose from your watchlist

### ANALYSIS
Comprehensive analysis tools
- **News Search** - Fetch latest market & company news
- **Technical Analysis** - RSI, MACD, moving averages, volume, price action
- **Fundamental Analysis** - P/E ratio, revenue growth, profit margins
- **Data Insights** - AI-powered data analysis
- **Live Chart** - Real-time price charts
- **Analyst Agent** - Comprehensive AI-powered analysis
